WebGL2 renderer.  
https://matteokeole.github.io/jsrenderer

Controls:
- [W]: Walk forward
- [S]: Walk backward
- [A]: Strafe left
- [D]: Strafe right
- [Space]: Fly up
- [LeftCtrl]: Fly down
